### Active Research

**[PAR-VAE](https://github.com/aaryaupadhya12/Physics-Attribute-Regularized-VAE)** — Physics Attribute-Regularized Variational Autoencoders  
Constraining 85-dim latent spaces to 14-dim physics attribute bases for interpretable 
medical AI. Quantifies the "Spatial Information Gap" as a principled performance bound.  

**[BGSAVM](https://github.com/aaryaupadhya12/BGSAVM)** — Motif-Grounded Uncertainty-Guided Spreading Activation  
Graph-based patch selection for 3D point cloud classification using evidential uncertainty 
and motif participation scores. Applies ViG backbone to unstructured 3D geometry.  

**[HetKGE](https://github.com/aaryaupadhya12/KG-EVOLVE)** — Heterogeneous KG Embeddings for Multi-Agent Routing  
Typed relational embeddings encoding structure between problems, agents, reasoning 
concepts and failure modes. Investigates compositional routing beyond flat vector similarity.
